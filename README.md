# Lab 4 – CRUD Catálogo

Aplicación React con CRUD completo (Crear, Leer, Actualizar, Eliminar) para un catálogo de usuarios.  
Basado en el ejemplo de la presentación TC3005B.

## Tecnologías
- React 18 (Class Component)
- Vite
- Bootstrap 5
- Reactstrap

## Cómo correr

```bash
npm install
npm run dev
```

Abre http://localhost:5173 en el navegador.
